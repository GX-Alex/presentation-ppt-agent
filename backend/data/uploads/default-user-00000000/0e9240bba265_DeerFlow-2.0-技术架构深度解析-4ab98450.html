错误码与故障排查

严重错误 — 需立即处理
警告提示 — 需关注检查
信息参考 — 正常状态说明
E1001
数据库连接失败
解决方案
检查网络连通性，确认数据库服务状态，验证用户名密码配置
E1002
权限认证异常
解决方案
重新登录系统，清除浏览器缓存，联系管理员重置权限
E1003
数据写入冲突
解决方案
刷新页面重试，等待30秒后操作，排查并发写入问题
W2001
响应时间过长
解决方案
检查服务器负载，优化查询语句，考虑增加缓存机制
W2002
存储空间不足
解决方案
清理临时文件，删除过期数据，扩容磁盘存储
W2003
接口调用限流
解决方案
降低请求频率，实现请求排队，等待限流恢复
I3001
缓存同步完成
说明
数据已成功同步至缓存层，系统运行正常
I3002
任务队列空闲
说明
后台任务已全部处理完成，无待执行任务
I3003
会话状态正常
说明
用户认证有效，会话未过期，可正常操作
