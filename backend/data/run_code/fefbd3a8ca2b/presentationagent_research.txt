# PresentationAgent 项目技术分享报告

## 一、项目概述

### 1.1 项目定位
PresentationAgent 是一个基于 AI Agent 的智能演示文稿生成平台，核心技术栈包括：FastAPI 后端、litellm 多模型 LLM 客户端、中间件链驱动的 Agent 运行时、WebSocket 实时通信、Next.js+React 前端、自研 Native Renderer PPTX 生成器。

### 1.2 技术架构（三层微服务）
- **Backend**: FastAPI + SQLAlchemy + 异步AIO
- **Frontend**: Next.js 14 + React + TypeScript + Tailwind
- **Native Renderer**: PPTXGENJS 渲染引擎

---

## 二、核心架构（AgentFactory + AgentRunner + MiddlewareChain）

### 2.1 AgentFactory (Agent 工厂)
- 文件：backend/app/core/agent_factory.py
- 职责：动态构建 AgentContext + MiddlewareChain
- 关键方法：create()、_build_chain()、create_subagent()
- 设计模式：按意图动态选择中间件、动态组装工具集、定制系统提示词

### 2.2 AgentRunner (Agent 运行器)
- 文件：backend/app/core/agent_runner.py
- 职责：执行 Agent 主循环
- 核心流程：消息持久化 → on_request_start → 组装上下文 → 主循环 → on_request_end

### 2.3 MiddlewareChain (中间件链)
- 文件：backend/app/core/agent_middleware.py
- 设计模式：洋葱模型（正序执行 before，反序执行 after）
- 关键钩子：on_request_start、on_before_llm、on_after_llm、on_tool_start、on_tool_end、on_round_end、on_request_end

### 2.4 工具分发机制 (Tool Dispatch)
- 文件：backend/app/core/tool_dispatch.py
- dispatch() 函数执行工具
- filter_tools_by_intent() 按意图过滤工具
- 工具分类：RESEARCH、PPT、CODE_ANALYSIS、MEMORY、MEDIA、UTILITY、UNIVERSAL

### 2.5 上下文管理
- 文件：backend/app/services/context_service.py
- assemble_context() 组装上下文
- compress_context() 压缩机制处理长对话
- MODEL_CONTEXT_WINDOW 默认 128K tokens

## 三、管理功能

### 3.1 Skill 系统
- 文件：backend/app/services/skill_service.py
- 系统预置 20+ 专业 Skills
- Skill 分类：PPTX文档编辑、DOCX文档编辑、研究分析师、代码审查专家、数据分析师、写作教练、会议主持人等

### 3.2 记忆系统
- save_to_memory / search_memory
- 长期记忆：偏好(preference)、事实(fact)、指令(instruction)、反馈(feedback)
- 用户画像构建与上下文注入

### 3.3 Plugin 架构
- Plugin Registry
- 动态 Tool 加载
- Package Runtime

## 四、智能工作区（四大核心能力）

### 4.1 WebDeck（PPT 演示文稿）
自研 Native Renderer PPTX 生成器，核心特性：
- WebSocket 实时通信，流式输出
- 页面状态管理：pending/generating/completed/failed
- 支持 8 种主题：tech_dark, ocean_gradient, warm_sunset, forest_green, royal_purple, minimal_gray, coral_energy, classic_blue
- Director-Planner-Scheduler-Reviewer 四角色协同架构
- 支持导出为可编辑 PPTX 格式

核心工具：
- generate_ppt_deck: 生成完整 PPT
- edit_deck_page: 修改指定页面
- retry_failed_deck_pages: 重试失败页面

### 4.2 Drawio 流程图/架构图
- 生成 draw.io 兼容的流程图/架构图
- 支持 Flowchart、UML 图、数据流图
- XML 格式输出
- 支持自定义样式和布局

### 4.3 Web 页面/应用生成
- HTML 网页/应用生成
- 可视化页面生成
- 响应式布局支持

### 4.4 智能文档
- Markdown 文档生成
- 研究报告输出
- 技术文档撰写

## 五、Docker 部署架构

### 5.1 三层微服务
- Backend (FastAPI + SQLAlchemy + 异步AIO)
- Frontend (Next.js 14 + React + TypeScript + Tailwind)
- Native Renderer (PPTXGENJS 渲染引擎)

### 5.2 通信机制
- 核心通信协议：WebSocket
- 二进制 PPTX 通过 Base64 传输
- 支持流式输出和实时进度反馈

### 5.3 前端状态管理
- Zustand 状态管理
- ChatStore 管理对话历史和流式内容
- DeckStore 管理 WebDeck 项目状态
- useWebSocket hook 实现全局单例 WebSocket（自动重连、心跳保活）

### 5.4 前端 AI 能力入口
- AI PPT 生成
- 调研报告
- 深度研究
- 网页搜索
- 代码分析
- 文案撰写
