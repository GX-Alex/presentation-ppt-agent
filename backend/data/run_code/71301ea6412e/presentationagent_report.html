📋 一、项目概述

PresentationAgent 是一个基于 AI Agent 的智能演示文稿生成平台，通过 AI 技术自动化演示文稿的创建过程。用户只需提供主题或内容需求，系统即可自动完成从内容研究、逻辑组织到视觉呈现的全流程工作。

🏗️ 二、核心架构（AgentFactory + AgentRunner + MiddlewareChain）

2.1 AgentFactory (Agent 工厂)

文件：backend/app/core/agent_factory.py

职责：动态构建 AgentContext + MiddlewareChain

create()
创建 AgentContext 和 MiddlewareChain
_build_chain()
根据意图组装中间件链
create_subagent()
创建子 Agent

2.2 AgentRunner (Agent 运行器)

文件：backend/app/core/agent_runner.py

职责：执行 Agent 主循环

// 核心流程
消息持久化 → on_request_start → 组装上下文 → 主循环 → on_request_end

2.3 MiddlewareChain (中间件链)

文件：backend/app/core/agent_middleware.py

设计模式：洋葱模型（正序执行 before，反序执行 after）

钩子 触发时机
on_request_start请求开始前
on_before_llm每轮 LLM 调用前
on_after_llm每轮 LLM 响应后
on_tool_start每次工具调用前
on_tool_end每次工具调用后
on_round_end每轮结束后
on_request_end请求完全结束后

2.4 工具分发机制 & 上下文管理

文件：tool_dispatch.py
dispatch() 执行工具
filter_tools_by_intent() 按意图过滤工具
load_skill() 动态加载 Skill
文件：context_service.py
assemble_context() 组装上下文
compress_context() 压缩长对话

六、总结

PresentationAgent 通过 AgentFactory + AgentRunner + MiddlewareChain 的核心架构，实现了灵活的 AI Agent 运行时系统。配合完善的 Skill 系统、记忆系统 和 多会话管理，支持 WebDeck、Drawio、Web 页面、智能文档 四大核心输出能力，为技术团队提供了一站式智能演示文稿生成解决方案。
