# Roadmap

PresentationAgent is moving toward a unified web-native deck runtime for research-heavy presentation work.

## Near Term

- Improve public onboarding and example projects.
- Add more sample WebDecks with source materials and exported artifacts.
- Improve editable PPTX export quality for chart-heavy decks.
- Tighten WebDeck page-level editing, save, rollback, and republish workflows.
- Document package and skill extension examples.

## Medium Term

- Stronger gallery publishing, fork, and remix workflows.
- More robust evidence tracing from source documents into generated pages.
- Better diagram session collaboration and draw.io recovery behavior.
- Export adapters with clearer quality modes: HTML, PDF, faithful PPTX, editable PPTX.
- Optional deployment guides for local, LAN, and server environments.

## Good First Issues

- Add a small example project under `examples/` with source files and expected output.
- Add a screenshot walkthrough for the WebDeck editing workflow.
- Improve `docs/PROMOTION.md` with one more launch channel.
- Add a troubleshooting note for Playwright Chromium export setup.
- Add a simple health-check script that validates backend and frontend prerequisites.

## Non-Goals

- Replacing PowerPoint as a full desktop editor.
- Generating unsupported claims without evidence.
- Treating decks as one-shot chat responses with no review or revision workflow.
