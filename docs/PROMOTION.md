# Promotion Kit

Use this file when announcing PresentationAgent in communities, social posts, newsletters, or adjacent projects.

## One-Liner

PresentationAgent is a local-first AI workspace for research-first presentation creation, WebDeck generation, diagram-first editing, and exportable deck assets.

## Short Description

Most AI presentation tools stop at prompt-to-slides generation. PresentationAgent is built as a production workspace: source materials flow through evidence-first planning, pages are generated through a dependency-aware WebDeck runtime, diagrams are edited through a draw.io-first workflow, and outputs can become assets, gallery items, or exportable presentation files.

Body:

```text
I open-sourced PresentationAgent, a local-first AI workspace for creating research-heavy presentations.

Instead of jumping straight from prompt to slides, it parses source material, prepares evidence-first briefs, orchestrates WebDeck pages with retries and review, supports diagram-first draw.io editing, and manages assets/gallery workflows.

It is built with FastAPI, Next.js, React, SQLite, WebDeck runtime, and local filesystem persistence by default.

GitHub: https://github.com/GX-Alex/presentation-ppt-agent
```
